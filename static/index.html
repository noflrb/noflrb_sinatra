Ruby Web Apps with Sinatra

North Florida Ruby Brigade

6 December 2011

Who is Sinatra?

  • An amazing singer and performer
  • Member of the 'Rat Pack'
  • Kickass, minimalist web framework

Hello World

$ gem install sinatra
# sinatra_demo/app.rb
require 'sinatra

get '/' do
  'Hello World!'
end
$ ruby -rubygems app.rb  # http://localhost:4567

Extending Hello World

Templates

<!-- sinatra_demo/views/index.erb -->
<html>
  <head>
    <title>Sinatra Demo</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>The time is <%= @current_time %>.</p>
  </body>
</html>
# sinatra_demo/app.rb
require 'sinatra'

class App < Sinatra::Application
  get '/' do
    @current_time = Time.now

    # Render views/index
    erb :index
  end
end
<!-- sinatra_demo/views/layout.html.erb -->
<html>
  <head>
    <title>Sinatra Demo</title>
  </head>
  <body>
    <%= yield %>
  </body>
</html>

<!-- sinatra_demo/views/index.erb -->
<h1>Hello World!</h1>
<p>The time is <%= @current_time %>.</p>
require 'sinatra'

class App < Sinatra::Application
  get '/' do
    @current_time = Time.now

    # Render views/index
    # Automatically pulls in layout.html.erb
    erb :index
  end
end

Development

$ gem install shotgun
$ shotgun app.rb

Development

  • Shotgun uses Unix model of forking for requests
  • Does not work on Windows or JRuby

Deployment

# config.ru
require './app'

run App
